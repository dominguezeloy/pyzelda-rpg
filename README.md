## About project 2.0

But now work in new vision of the game, the future is amazing, the new possibilities is like a horizont.

## Run project locally

```bash
git clone https://github.com/artemshchirov/pyzelda-rpg.git
cd pyzelda-rpg
pip install pygame
python code/main.py
```
